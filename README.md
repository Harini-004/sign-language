# Deep Learning-Based Gesture Recognition System

This project is a deep learning-based system designed to enable two-way communication for deaf and speech-impaired individuals by recognizing hand gestures and also converting text into sign representations.

## Features
- Recognizes hand gestures using a trained deep learning model
- Converts gestures into meaningful text output
- Converts text input into corresponding sign representations (two-way communication)
- Focused on improving accessibility and communication

## Tech Stack
-Python
-Django
-OpenCV
-MediaPipe
-TensorFlow/Keras
-NLTK
-NumPy

## How It Works
The system captures hand gestures and processes them using a trained deep learning model to predict outputs. It also supports reverse communication by converting text into sign-based representations, enabling two-way interaction.
